# @dbx-tools/ui-appkit-client

Minimal AppKit UI example.

## Overview

This package demonstrates a React/Vite package that depends on
`@databricks/appkit` and `@databricks/appkit-ui`. It is intentionally small so it
can compile without a live Databricks workspace.

## Usage

Run from this package directory:

```sh
pnpm dev
```

## Module

- `App` - a placeholder AppKit client component.

For fuller Databricks App UI patterns, use the official AppKit UI documentation
and the `@dbx-tools/ui-appkit` shared foundation package.
