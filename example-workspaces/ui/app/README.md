# @dbx-tools/ui-app

Minimal React/Vite example package.

## Overview

This package demonstrates the `ui` tag defaults: React, React DOM, Vite,
DOM-aware TypeScript settings, and `dev` / `preview` tasks.

## Usage

Run from this package directory:

```sh
pnpm dev
```

## Module

- `App` - a small React component exported through the generated barrel.

Use this package to validate the generic UI toolchain, not AppKit-specific UI
integration.
