# @dbx-tools/cli-main

Example CLI package generated by the `cli` tag.

## Overview

This package ships the `pw-demo` command. It uses `commander` for argument
parsing, `@clack/prompts` for terminal banners, and the example shared package
for arithmetic.

## Usage

Run from this package directory:

```sh
pnpm pw-demo 1 2 3
```

Or import the command runner from the generated barrel:

```ts
import { cli } from "@dbx-tools/cli-main";

const total = cli.run(["1", "2", "3"]);
```

## What it demonstrates

- `cli` tag defaults: Node types, `commander`, `@clack/prompts`, and bin wiring.
- Workspace dependency wiring to `@dbx-tools/example-shared-core`.
- Generated package-root barrel exports.
