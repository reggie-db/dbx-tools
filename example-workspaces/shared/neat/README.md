# @dbx-tools/shared-neat

Tiny shared example package.

## Overview

This package demonstrates that a source-bearing folder under
`example-workspaces/shared` becomes a workspace package and receives a generated
barrel.

## Module

- `cool` - exports `coolDude()`.

## Usage

```ts
import { cool } from "@dbx-tools/shared-neat";

cool.coolDude();
```
