Terminal / Prompt-Cursor logo concepts

Four marks built from developer-terminal motifs — the >_ prompt, a blinking cursor block, </> code brackets, and a terminal window. Clean geometric sans wordmark (DM Sans, à la Databricks) in a warm iPhone-Coral palette. Each mark hides an extremely subtle mustache — invisible until you know, delightful once you do.

concept 01 · prompt

The >_ Prompt

The universal shell prompt, reduced to its essence: an open chevron and a resting cursor. The most instantly-legible “this is a terminal” signal, and the strongest favicon of the set.

🥸 Mustache: the two arms of the > are gently waxed — curved outward like a groomed handlebar meeting at the point (the philtrum). Reads as a crisp chevron; only on a second look is it a tiny mustache.

concept 02 · cursor

The Cursor Block

A solid blinking cursor resting on the command line — the beat of an active session. The most abstract and monogram-like of the set; the tall block pairs naturally with the ascenders of the wordmark.

🥸 Mustache: the command-line baseline isn’t flat — its two tips flick upward a couple of pixels, a barely-there handlebar under the cursor. Looks like a cursor on a line; it’s quietly a face.

concept 03 · brackets

The </> Code Brackets

The “this is code” glyph, dressed in coral. Two chevrons split by a slash — symmetrical, energetic, and unmistakably about software tooling. Strong horizontal balance beside the wordmark.

🥸 Mustache: read the two chevrons as the halves of a handlebar and the angled / becomes the nose-and-philtrum parting them down the middle. It’s a code tag first; a face on the second glance.

concept 04 · window

The Terminal Window

The whole environment in one glyph: a rounded window, a title bar, and a live >_ prompt inside. The most “product” of the set — an app-icon shape that still nods to the command line.

🥸 Mustache: the prompt > inside the window has the same waxed, curling arms as Concept 01 — a mustache tucked behind the glass. Doubly hidden: you have to notice the prompt first, then the mustache within it.
