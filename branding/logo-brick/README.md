# dbx.tools Logo Concepts — Brick / Pixel-Block Lane

Phase 1 concept set for `dbx.tools`, focused on a nerdy developer-tool aesthetic with pixel bricks, stacked blocks, and subtle Databricks-adjacent geometry without copying the Databricks logo.

## Shared System

- **Palette:** primary coral `#F86C5A`, hot coral `#FF7A5C`, dark coral `#C94E44`, light tint `#FFF0EA` / `#FFF7F4`, dark surface `#0A0A0A`.
- **Typography:** `DM Sans` is specified first for the wordmark to stay near the Databricks geometric sans feel, with `Inter` and system sans fallbacks.
- **Format:** each concept includes light and dark SVG variants for both icon-only and horizontal lockup files; unsuffixed `concept-N-icon.svg` and `concept-N-lockup.svg` files are light-mode aliases for quick previewing.
- **Contact sheet:** open `index.html` directly in a browser; it references only local files and needs no server.

## Concepts

### 1. Pixel d Stack

- **Files:** `concept-1-icon-light.svg`, `concept-1-icon-dark.svg`, `concept-1-lockup-light.svg`, `concept-1-lockup-dark.svg`.
- **Idea:** a lowercase `d` assembled from crisp coral pixel bricks, optimized to remain recognizable at 16px and 32px favicon sizes.
- **Mustache easter egg:** two tiny negative-space rounded notches are cut into the right-side bowl bricks of the `d`; they read as pixel gaps unless pointed out.

### 2. Isometric Build Blocks

- **Files:** `concept-2-icon-light.svg`, `concept-2-icon-dark.svg`, `concept-2-lockup-light.svg`, `concept-2-lockup-dark.svg`.
- **Idea:** a small isometric stack of coral cubes, nodding to stacked data/tooling blocks while keeping the silhouette distinct from the Databricks mark.
- **Mustache easter egg:** the thin curved cut between the two lower cubes creates a barely visible mustache silhouette in the central negative space.

### 3. Block Wrench

- **Files:** `concept-3-icon-light.svg`, `concept-3-icon-dark.svg`, `concept-3-lockup-light.svg`, `concept-3-lockup-dark.svg`.
- **Idea:** a pixel-built wrench/tool glyph for the package/tooling story, using square bricks and a diagonal handle for a more utility-forward direction.
- **Mustache easter egg:** two adjacent negative-space notches are hidden in the wrench jaw gap, doubling as the wrench opening at small sizes.

### 4. Terminal Brick Cursor

- **Files:** `concept-4-icon-light.svg`, `concept-4-icon-dark.svg`, `concept-4-lockup-light.svg`, `concept-4-lockup-dark.svg`.
- **Idea:** a blocky terminal chevron with a coral cursor brick, leaning into CLI, packages, and retro-digital developer ergonomics.
- **Mustache easter egg:** two small negative-space pixels inside the cursor brick form a tiny mustache-like pair of dots without becoming a face.
